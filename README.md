# Escape the Haunted House

## What does this app do?
Escape the Haunted House is a simple, spooky game where you try to escape a haunted house before time runs out. The game gets harder and scarier with each level.

## How to run locally
1. Download or clone the repo.
2. Open the `index.html` file in your web browser.

## How to deploy to Netlify
1. Create a new repository on GitHub (e.g., "escape-the-haunted-house").
2. Copy all files from the repo and push them to GitHub.
3. Go to [Netlify](https://app.netlify.com/), click **Add New Site**, and import your GitHub repo.
4. Netlify will auto-deploy the site to a live URL.

## How to customize
- **Game settings**: Modify the constants in `config.js`.
- **Edit content**: Change text in `index.html`, and adjust colors/styles in `style.css`.
